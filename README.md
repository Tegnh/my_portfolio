# معرض أعمال طارق الفضل | Tariq Al-Fadhel Portfolio

موقع شخصي (Portfolio) يستعرض المهارات، المشاركات، والمشاريع البرمجية المنشورة في مجالات الذكاء الاصطناعي، تطوير الويب (Full-Stack)، والأعمال والتمويل.

## 🛠️ التقنيات المستخدمة
* **الهيكل والواجهة**: HTML5, Vanilla CSS3, JavaScript
* **الخطوط**: Cairo, Outfit (Google Fonts)
* **المؤثرات**: Scroll Reveal, CSS Grid, Flexbox
